El proyecto propone la implementación de un Sistema de Observador del Aprendiz en el Centro Tecnológico de la Producción Industrial (CTPI) del SENA con el fin de fortalecer el seguimiento formativo y mejorar la comunicación entre instructores, coordinación académica y aprendices. La iniciativa responde a la necesidad de centralizar la información relacionada con el proceso formativo del aprendiz y facilitar la identificación temprana de dificultades académicas o comportamentales. Mediante el uso de herramientas digitales y protocolos institucionales de registro formativo, el sistema permitirá mejorar la gestión académica y promover un acompañamiento pedagógico más efectivo.
